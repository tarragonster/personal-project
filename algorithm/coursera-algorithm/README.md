### Java Note

#### 1. compile java file to class file with local lib
```shell
javac -cp .:./libs/algs4.jar [filename].java
```

#### 2. run compiled class file with local lib
```shell
java -cp .:./libs/algs4.jar [filename]
```

#### 3. End of file (EOF)
- Control D
