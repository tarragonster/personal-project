# Nodejs-tcp Server Demo

tcp require connection (connection based)
stateful (client and server know each other)

## Start Nodejs docker

```
cd webserver
docker-compose up -d nodejs-tcp
```

## Run Demo

```
open local terminal
command -> telnet 0.0.0.0 14
type hello
```
